# API

API schemas, OpenAPI specifications, and contracts for the SLN AI System.

## Overview

Centralized API definition layer for all system interfaces.

## Structure

```
api/
├── openapi/           # OpenAPI 3.0 specifications
├── graphql/           # GraphQL schemas
├── grpc/              # Protobuf definitions
├── contracts/         # API contracts & versioning
└── mocks/             # Mock API responses
```
