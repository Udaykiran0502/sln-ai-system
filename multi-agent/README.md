# Multi-Agent

Multi-agent coordination, communication, and collaboration framework.

## Overview

Manages inter-agent communication, task delegation, consensus mechanisms, and collaborative workflows.

## Patterns

| Pattern | Description |
|---------|-------------|
| Pipeline | Sequential agent chain |
| Fan-out/Fan-in | Parallel execution with aggregation |
| Supervisor | Central coordinator with worker agents |
| Debate | Adversarial agents for consensus |
| Swarm | Autonomous agent collaboration |
